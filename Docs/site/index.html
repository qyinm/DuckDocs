Three steps to perfect docs

From screen capture to polished documentation in seconds, not hours.

Smart Capture

Select any region, window, or full screen. Configure your "next" action and let DuckDocs auto-capture each step of your workflow.

  • Region, window, or full screen
  • Configurable capture intervals
  • Keyboard action automation

AI Processing

Vision AI analyzes each capture in parallel, understanding context, UI elements, and generating accurate descriptions.

  • Parallel image processing
  • Multiple AI providers supported
  • Context-aware descriptions

Markdown Export

Get beautifully formatted markdown with embedded images, ready for GitHub, Notion, or any documentation platform.

  • Clean, semantic markdown
  • Organized image folders
  • Copy or export anywhere
